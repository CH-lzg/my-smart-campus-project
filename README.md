# my-smart-campus-project
# 🎓 校园智慧服务与数据分析管理系统
一个面向高校的综合性数字化管理平台，集**学生信息管理、教务管理、数据分析、校园服务**于一体，为学校提供智能化、数据化的管理解决方案。

## ✨ 项目简介
本项目是一套轻量级校园服务管理系统，基于 Flask 后端 + HTML/CSS/JavaScript 前端开发，实现了管理员端与学生端双角色的完整业务流程。
- **管理员端**：实现学生信息、课程、成绩的全生命周期管理与数据统计分析
- **学生端**：提供课程查看、成绩查询、个人信息管理等服务
- 系统界面简洁清晰，适配校园管理场景，去除了模板化设计，还原真实系统使用体验

- ## 🚀 功能模块
### 🔐 双角色系统
| 角色 | 核心能力 |
|------|----------|
| **管理员端** | 学生信息增删改查、课程管理、成绩录入、数据分析仪表盘 |
| **学生端** | 个人课程查看、成绩查询、个人信息管理、校园服务入口 |

### 📊 核心功能
- **仪表盘数据看板**
  - 管理员端：学生总数、课程总数、平均分、及格率实时统计
  - 学生端：选修课程数、平均成绩、及格率、校园服务快捷入口
- **数据分析模块**：成绩分布直方图、课程平均分对比图表，直观呈现学习数据
- **课程管理**：管理员课程创建与维护、学生课程查看与选课状态管理
- **成绩管理**：管理员批量录入/修改成绩、学生实时查看个人成绩
- **个人中心**：用户信息查看、修改，安全退出登录

- ## 🛠️ 技术栈
| 类别 | 技术选型 |
|------|----------|
| 后端 | Python / Flask |
| 数据库 | MySQL |
| 前端 | HTML / CSS / JavaScript |
| 图标库 | Font Awesome |
| 数据可视化 | ECharts |
